About

This service provides one place for analysts to access the things they need to analyse and model data.

  1. Find data - links to the Data Discovery platform, a new product which includes a catalogue of metadata and data sets available. Not all data is available yet.
  2. Analyse data - links to Databricks, the access layer to enable analysis and modelling of data.
  3. Request data - links to a ServiceNow form that allows users to request access to datasets in AnM, EDAP1 and PDR
  4. Get support - that's this section - with learning materials for Databricks and links to the communities on MS Teams that can provide support when you need it.
  5. Published reports - links to reports published on Explore Education Statistics public portal and to a repository of internal shared reports.
  6. News and updates - view information on the latest data and feature updates.

Get started

The ADA service is available to anyone from the Department of Education as long as they're connected to the DfE network.

To request data via the Data Discovery platform, you need to log in using your DfE email.

To request access to analysis and modelling data via Databricks, you will need to raise a request on the IT Service Portal. You can do this via the “Request data” feature from the ADA homepage. To access the service you will need to set up two-factor authentication.


Roadmap

The ambition is to incrementally enable access to all datasets across DfE and ESFA and retire the legacy presentation layer applications, enabling common ways of working and collaboration between teams across DfE / ESFA areas.

The Data Discovery and 'Published reports' areas that link to our service will continue to evolve, providing a joined-up view of data across the Department.
