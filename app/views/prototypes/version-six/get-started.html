Getting started with Databricks

If you haven't used Databricks before - you will need access to CIP and the Databricks platform.

We are moving datasets in stages. In time - all datasets will be available on Databricks. See the datasets available in databricks now.


Make sure you follow our Best practice rules for working with data in Databricks.

Analysing data using Databricks

You can choose which method to use - depending on your previous experience.

Databricks Notebooks

Notebooks is the simplest and most intuitive way to work with data for analysts new to the platform. It provides real-time co-authoring in multiple languages, automatic versioning, and data visualisation.

Databricks SQL Editor

SQL Editor is similar to SQL Server Management Studio (SSMS). It provides a simple interface to run queries on data using SQL.
