Analysing data using Databricks

You can choose which method you want to use - depending on your previous experience.

Databricks Notebooks

Notebooks is the simplest and most intuitive way to work with data for analysts new to the platform. It provides real-time co-authoring in multiple languages, automatic versioning, and data visualisation.


Databricks SQL Editor

SQL Editor is similar to SQL Server Management Studio (SSMS). It provides a simple interface to run queries on data using SQL.


RStudio on Databricks

RStudio is an Integrated Development Environment (IDE) for users of R. Databricks offers the functionality to run a hosted version of RStudio allowing analysts access to all data and the distributed compute resources of the Databricks clusters.


VS code with Databricks extension

VS Code is an Integrated Development Environment (IDE) used by many software developers. For anyone familiar with this IDE, Databricks offers an extension to allow interaction with data held in the Databricks Platform.
