Using SQL in Databricks SQL Editor

Create and construct a query in SQL Editor

On the left menu click on SQL to expand the menu and select SQL Editor.


You are now on the SQL dashboard.
In the data warehouse dropdown select SQL Warehouse Compute.


The icon next to the SQL Warehouse Compute shows the status.

Running

Starting

Stopped


Click on the Run button to start your run.

You can filter the list by typing in the filter field.


Start a warehouse

To manually start a stopped SQL warehouse click on SQL Warehouses in the left sidebar.

Click on the start icon in the Status column - next to the warehouse.

If a SQL warehouse is stopped and you attempt to run a job or query that uses it - Databricks starts the warehouse automatically.


A warehouse also restarts automatically if you open a query in the SQL editor that is saved to a stopped warehouse or if you open a dashboard that is saved with a dashboard-level warehouse assigned to it.



Auto-restart works for all types of SQL warehouses, including serverless SQL warehouses, which start very quickly.
