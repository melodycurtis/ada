Using SQL in Databricks SQL Editor

Create and construct a query in SQL Editor

On the left side bar click on the dropdown and select SQL.


Click on SQL Editor.


Create a query

You should see the SQL warehouse dashboard.
By default, warehouses are sorted by state - running warehouses first - then in alphabetical order.

Click New in the sidebar and select Query.

Type SELECT in the box and start typing. The browser will suggest databases and tables that are available to you.




Consruct a query

You can construct a query by inserting elements from the schema browser or typing in the SQL editor.

  • To insert an object from the schema browser into the SQL editor, click the double arrow on the right of a data object.
  • Type wer query in the SQL editor.

SQL editor supports autocomplete. As you type, it suggests completions. For example, if a valid completion at the cursor location is a column, it wil suggest a column name. If you type select * from table as t where t., autocomplete recognizes that t is an alias for table and suggests the columns inside table.
