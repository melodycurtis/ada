Using R, Python and SQL in Databricks

Using R, Python and SQL

Create a notebook in your workspace

Sign in to Databricks and make sure you are in the Data Science and Engineering environment.

You can change the environment on the left sidebar.


To find your personal folder select Workspace from the sidebar. Then select the Users folder from the menu.

In the Users folder you will see your username at the top. This is your personal folder - no one else will have access to it. A sandpit area - where you can test things.

Create a Notebook in this folder.


Create a Notebook

  1. Click the dropdown next to your name and select Create and Notebook.


  2. Your new Notebook will open. It will be given a default name and coding language.


  3. Edit the name of the Notebook by selecting it in the top bar. Select a language by clicking on the dropdown on the right. Select a cluster by clicking on the Connect button.


  4. Type some code in to the command cell to check that everything is running.


    5. Contents

    • Python
    • R
    • SQL

    print("Hello World")

    print("Hello World")

    SELECT "Hello World"

  5. Click on the Run all button on the top bar to run your code.
  6. You can add additional code cells by clicking on the plus icon + at the bottom. This will help you to structure large amounts of code and make it easier to navigate.
