Create a notebook in your workspace

Select Workspace from the side menu and select Workspace and Users.

This is your personal folder - no one else will have access to it. Use this area to test things out.

Create a notebook in this folder.


Create a Notebook

  1. Click on the dropdown button on the right and select Notebook.


  2. Your new notebook will be called 'untiltled' and will have a default coding language. Click on the name to edit it and select a language by clicking on the dropdown


  3. Click on the dropdown on the top right - to select your cluster. It might say Connect or Terminated. The example below is connected to the ADA Project cluster.

    4. If you're using Python or SQL - you should use a shared cluster.
    If you're using R - you need to create a personal cluster.

    It's important to know how clusters work


  4. Type your code in the command cell. Or select the tab with the language you want to use and click on the copy code button.


  5. Use the tabs below to try out some code. Select the language you want to use and click on the copy code button. Paste into your command cell.

    6. Contents

    • Python
    • R
    • SQL

  6. Click on the Run all button on the top bar to run your code.

    7. You may need to wait for your cluster to start


  7. You can add code cells if you hover near the bottom of a cell and click on the plus icon. This will help to structure large amounts of code and make it easier to navigate.
