Notebooks - loading data

Find your data

You need to know the path to your data - to be able to load it.


Select Data from the left sidebar.

Make sure you open it in a new tab or window - so you can carry on working in your Notebook.


  1. The environments you have access to will be displayed here. The datasets will sit in the environment catalogue. Pre prod and Prod is where the real data sits.


  2. Enter a name and select a language. Select a cluster. If you can't see one - you can create one later. Click on Create.


  3. Your new Notebook will open - and you can start working.


  4. Click on the Run all button on the top bar to run your code.
