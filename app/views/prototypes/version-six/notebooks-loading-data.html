Notebooks - loading data

Find the path to your data

Before you start - you will need to know the path to your data to be able to load it. This will be a three part name with a catalog, schema and table.

Example: preprod_catalog_p02.ada_demo.csc_data

Select Data from the left sidebar - but make sure to open it in a new tab or window - so you can carry on working in the Notebook you've created.


  1. The catalogs you have access to will be displayed here. Pre prod and Prod is where the real data sits. You may want to work in Preprod using real data - but not affecting the live service.


  2. You will see schemas in each catalog. Select the schema in your path.


  3. Find your table and select it. Then copy the path of the data you've selected.


  4. Go back to your Notebook in the other tab or window that says 'Hello World'.


  5. Paste the path of your data into the cell.
  6. Type in the load commands related to your language and copy the path to your data in to the cell.
