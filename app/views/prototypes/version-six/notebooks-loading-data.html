Notebooks - loading data

Find your data

You need to know the path to your data - to be able to load it.

Like this: preprod_catalog_p02.ada_demo.csc_data

Select Data from the left sidebar - but make sure you open it in a new tab or window - so you can carry on working in the Notebook you've created.


  1. The catalogues you have access to will be displayed here. Pre prod and Prod is where the real data sits. You may want to work in Preprod using real data - but not affecting the live service.


  2. You will see schemas in each catalogue. Select the schema in your path.


  3. Find your table and select it. Then copy the path of the data you've seleted.


  4. And go back to your Notebook in the other tab or window that says 'Hello World'.


  5. Hover over the bottom of the cell and click on the plus + icon to create a new cell.

  6. Paste the path of your data into the cell.
  7. Type in the commands related to your code and run it.
