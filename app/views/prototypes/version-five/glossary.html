List of terminology

Term Definition
Analyst Workbench The toolkit that allows analysts to perform analysis and modelling of data. This is the internal functional name for Databricks, a commercially licensed cloud-based platform.
Analytical Data Access (ADA) The internal DfE service that allows staff to find data and access analytical tools.
Data sets A collection of related sets of information. For example - a dataset about pupil attendance which includes pupil, school, and attendance information sets.
Databricks A cloud based data engineering and analytics platform which DfE has licensed. A suite of tools to support data engineers and analysts to store data and tooling - to query and analyse large data sets.
Delta file The format of the way data is stored in the cloud-based storage delta lake. A delta file compresses information and only makes changes to the parts of the file that have changed when you need to copy, download or upload it.
Delta lake A storage layer that provides the foundation for storing data and table in the Databricks platform. Delta lake is open source software and it is the default storage format for all operations on Databricks.
Metadata A set of data that describes and information about other data. For example school metadata might include the school name, identifier number, address.
Private beta The stage of delivery in a project or service where the service is available to a limited number of users - so that it can be thoroughly tested and improved until it is released to a wider audience. This is part of the gov.uk Service Standard.
Python A programming or coding language which the analyst community use to develop their models for analysis. It is widely used in data analysis and data science for importing, analysing and visualisation of data.
R A programming or coding language and environment which the analyst community use to develop their models for analysis. It has features which are useful for statistical analysis and graphical representation.
Reports and dashboards A feature in the ADA service that provides a library of reports and dashboards that can be used by DfE staff to find and query data. In some cases - access will be restricted to authorised users only. The ambition is to grow this library over time. This area also provides navigation to the Explore Education Statistics public-facing portal.
SQL Structure Query Language (SQL) is a programming or coding language for use in data and database manipulation.
