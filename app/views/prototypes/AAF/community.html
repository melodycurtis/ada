About

This service provides one place for analysts to access the systems they need to analyse and model data.
Find data Link to the new Data Discovery platform. A catalogue of metadata and data sets. There are two sections Data Governance and Data Marketplace.
Analyse data Link to Databricks. The access layer to enable the analysis and modelling of data. You will need to log in with your DfE details.
Request data Links to a ServiceNow form - where you can request access to datasets in ADA, AnM, EDAP1 and PDR.
Get support Links to learning materials for Databricks and the communities on MS Teams that can provide support when you need it. That's this section.
Published reports Links to reports published on the Explore Education Statistics public portal and to a repository of internal shared reports.
News and updates The latest news plus any new data and feature updates.

Get started

The ADA service is available to anyone from the Department of Education as long as they're connected to the DfE network.

To request data via the Data Discovery platform, you need to log in using your DfE email.

To request access to analysis and modelling data via Databricks, you will need to raise a request on the IT Service Portal. You can do this via the “Request data” feature from the ADA homepage. To access the service you will need to set up two-factor authentication.

Roadmap

The ambition is to incrementally enable access to all datasets across DfE and ESFA and retire the legacy presentation layer applications. This will create common ways of working and collaboration between teams across DfE / ESFA areas.

The Data Discovery and 'Published reports' areas that link to our service will continue to evolve, providing a joined-up view of data across the Department.
