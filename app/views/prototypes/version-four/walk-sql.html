Getting started with SQL Editor

First steps

Sign in to Databricks and make sure you are in the SQL workspace by selecting SQL on the left sidebar.

This will change the menu items.

On the left sidebar again - select SQL Editor.

This takes you straight in to a working space. It will look similar to Microsoft SQL Server Management Studio (SSMS).

Run a query

  1. On the top bar on the right side select a SQL warehouse from the drop down. This may take a few minutes to load.


  2. On the top bar on the right side select a SQL warehouse from the drop down. This may take a few minutes to load.


  3. Click on the Run all button on the top bar to run your code.
