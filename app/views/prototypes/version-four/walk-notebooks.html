Getting started with Databricks Notebooks

Requirements

When you're logged in to Databricks - make sure you're in the 'Data Science and Engineering' workspace. Select the workspace type from the top left sidebar.


Create a Notebook

  1. Using the workspace directory menu, navigate to where you want to create your new Notebook. A good starting place is your user home directory. This is private to you by default - although you can share Notebooks if you want to.

  2. In the Workspace folder select Create > Notebook.
  3. On the 'Create Notebook' dialog enter a name and select SQL in the 'Default Language' drop-down. This will determine the language of the notebook.

  4. Click Create.
  5. The Notebook opens with an empty cell at the top.

  6. Attach the Notebook to an available cluster.


Workspace root folder

The workspace is the special root folder that stores your Azure Databricks assets, like notebooks and libraries, and the data you import.

To navigate to the Workspace root folder:

  1. Click on Workspace.
  2. Click on the drop-down arrow.

The Workspace root folder is a container for all of our organisation's Azure Databricks static assets.


Within the Workspace root folder:

  • Shared is for sharing objects across our organization. All users have full permissions for the objects in Shared.
  • Users contains a folder for each user.



Create a Notebook in any folder

We can create a new Notebook in any folder. For example - in the Shared folder, following these steps:

  • In the sidebar, click Workspace. Do one of the following:

    • Next to any folder, click the on the right side of the text and select Create > Notebook.

    In the workspace or a user folder, click and select Create > Notebook.

  • Follow steps 2 to 4 in Use the Create button.

How to copy a Notebook path

To copy a notebook file path without opening the notebook, right-click the notebook name or click the to the right of the notebook name and select Copy File Path.



Rename a Notebook

To change the name of an open notebook, click and edit inline or click File > Rename.



How to Switch to a different workspace

If you have access to more than one workspace in the same account, you can switch from one to the other.

  1. Click the workspace name in the top bar.
  2. Select a workspace from the drop down to switch to.



Code languages in Notebooks

Click the drop down next to the Workbook name and select a language.
