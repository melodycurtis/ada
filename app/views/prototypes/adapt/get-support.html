Accessing data

All data access is managed by Service Now. Over time you will be able to request access in the Data Discovery Platform or the Databricks application.

Data Sharing Agreements

You need a Data Sharing Agreement before you can access data.

If you already have one it will need to include permission to access data using Service Now. Once approved you will be issued with a username.
To discuss access call the support team on 0300 303 333 or email enquiries@education.gov.uk
