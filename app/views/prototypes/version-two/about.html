How to use this service

This service provides one place for analysts to access the systems they need to analyse and model data.

It's available to anyone who works for the Department of Education as long as you're connected to the DfE network.



The future

The ambition is to enable access to all datasets across DfE and ESFA and retire the legacy presentation layer applications. This will create common ways of working.
Find data Links to the Data Discovery Platform. The Data Marketplace section is a catalogue of metadata and data sets. Log in using your DfE email.
Analyse data Analyse and model data using Databricks - you'll need to raise a request for access on the IT Service Portal. Do this via the “Request data” tile. You will need to set up two-factor authentication.
Request data Links to the ServiceNow form - where you can request access for datasets in ADA, AnM, EDAP1 and PDR.
Get support Links to learning materials for Databricks and the communities on MS Teams that can provide support when you need it. That's this section.
Published reports Links to reports published on the Explore Education Statistics public portal and to a repository of internal shared reports.
News and updates The latest news plus any new data and feature updates.
